# odin-recipes
A website consisting of a home page which links a few recipes. The recipes currently added are the following:

- Carbonara
- Sourdough
- Scrambled Eggs

Each recipe page links back to the home page and contains an image, a description, a list of ingredients, and a list of steps to make the food in the recipe.

This project was built as a learning exercise to practice using common HTML elements, attributes, and understanding the use of relative links.
